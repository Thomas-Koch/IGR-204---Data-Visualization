Instructions to start the project correctly

  1. Download and unzip the contents of the France.zip folder into the directory of your choice.
  2. Start a local http server on port 8080. To do this, open a terminal and copy the line below, replacing $PATH$ with the path to the directory where you have the France folder.
cd $PATH$/France/ && python -m http.server 8080
  1. In a browser, go to the following address: http://localhost:8080/

First visualization

For this first visualization, I simply followed the instuctions requested in the TP statement. So we get a map of France with ** displaying information about the city overflown with the mouse**. This information is displayed above the map.

Second display

For this second visualization, I started again from the previous work by making the map more interactive with a dynamic display based on population density.

It is thus possible to zoom and/or move the map with the mouse. As before, flying over a city allows you to display its information above the map. I also added a style effect since the selected point changes color and enlarges.

Note : Corsica appears badly here because of its low population density compared to other regions of France.
